# SnapAI

Браузерное расширение для Chrome/Edge, которое добавляет AI-ассистента прямо на любую веб-страницу. Выделите текст — получите объяснение, резюме или развёрнутый анализ от AI.

Поддерживает два провайдера: **OpenAI** и **OpenRouter** (доступ к сотням моделей через один ключ).

---

## Возможности

- Выделите любой текст на странице → появится иконка **✦**
- Кликните на иконку → откроется панель с тремя быстрыми действиями:
  - **Объяснить** — простое объяснение текста
  - **Резюмировать** — краткое изложение
  - **Разобрать подробнее** — развёрнутый анализ
- Задайте свой вопрос в поле ввода
- Диалог сохраняется в рамках сессии (до 5 пар сообщений)
- Скопируйте ответ одной кнопкой
- Закройте панель нажатием **Escape** или кликом вне панели

## Требования

- Node.js 18+
- npm
- Chrome 120+ или Edge 120+

## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone <repo-url>
cd <repo_name>
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Соберите расширение

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

### 4. Загрузите расширение в браузер

1. Откройте `chrome://extensions` (или `edge://extensions`)
2. Включите **Режим разработчика** (Developer mode) — переключатель в правом верхнем углу
3. Нажмите **Загрузить распакованное** (Load unpacked)
4. Выберите **корневую папку** проекта (там где лежит `manifest.json`)

---

## Настройка

1. Кликните правой кнопкой на иконку расширения → **Параметры** (или через `chrome://extensions` → Details → Extension options)
2. Выберите провайдера: **OpenAI** или **OpenRouter**
3. Введите API-ключ
4. При необходимости укажите модель (или оставьте по умолчанию)
5. Нажмите **Сохранить**

### OpenAI

- Ключ получить: https://platform.openai.com/api-keys
- Модель по умолчанию: `gpt-4o-mini`
- Формат ключа: `sk-...`

### OpenRouter

- Ключ получить: https://openrouter.ai/keys
- Модель по умолчанию: `openai/gpt-4o-mini`
- Полный список моделей: https://openrouter.ai/models
- Формат ключа: `sk-or-...`

> OpenRouter даёт доступ к моделям OpenAI, Anthropic, Google, Meta и другим через один API-ключ.

## Использование

1. Откройте любую веб-страницу
2. Выделите текст мышью
3. Подождите ~300-350 мс — появится фиолетовая иконка **✦** справа от выделения
4. Кликните на иконку
5. В открывшейся панели:
   - Нажмите одну из кнопок действия, или
   - Введите свой вопрос и нажмите **Отправить** (или Enter)
6. Для закрытия панели нажмите **Escape** или кликните вне неё

## Структура проекта

```
├── src/
│   ├── background.ts   # Service Worker: обработка AI-запросов, маршрутизация к провайдеру
│   ├── content.ts      # Content Script: отслеживание выделения, иконка-триггер
│   ├── panel.ts        # UI панели (Shadow DOM)
│   ├── options.ts      # Страница настроек
│   └── types.ts        # Типы, константы, схема хранилища
├── options.html        # HTML страницы настроек
├── manifest.json       # Манифест расширения (MV3)
├── build.js            # Скрипт сборки (esbuild)
└── dist/               # Собранные файлы (генерируется автоматически)
```

## Хранилище данных

Все настройки хранятся в `browser.storage.local`:

| Ключ               | Описание                                      |
|--------------------|-----------------------------------------------|
| `provider`         | Выбранный провайдер: `openai` или `openrouter` |
| `openaiApiKey`     | API-ключ OpenAI                               |
| `openrouterApiKey` | API-ключ OpenRouter                           |
| `openaiModel`      | Модель OpenAI (по умолчанию `gpt-4o-mini`)    |
| `openrouterModel`  | Модель OpenRouter (по умолчанию `openai/gpt-4o-mini`) |

Если вы использовали расширение до добавления поддержки OpenRouter, ваш старый ключ (`apiKey`) автоматически мигрирует в `openaiApiKey` при первом запросе.
